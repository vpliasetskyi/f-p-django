Search & Discovery: Integrates with the TMDB API so users can look up titles.

Custom Lists: Users can create personal watchlists, add custom ratings, and write personal comments/reviews for each item.

Social Connectivity: Users can share their custom lists and seamlessly save other users' lists directly to their own profiles.

Custom Additions: If a movie/show is missing from TMDB, users have full CRUD capabilities to manually input details (Title, Year, and upload a Poster Image).
